# 基于协同过滤推荐算法的音乐推荐系统的设计与实现

#### Description
基于协同过滤推荐算法音乐推荐系统，推荐算法使用协同过滤推荐算法，后端框架使用Springboot+mybatis，前端框架使用Vue，数据库使用mysql，系统分为前台用户使用系统和后台管理员登录系统和用户管理登录系统三个系统。
